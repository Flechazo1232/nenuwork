# 时间规划器

一个美观、直观的时间规划工具，帮助您高效管理一天24小时的任务。

## 功能特点

- **环状时间图**：以24小时环形时钟的形式展示您的一天
- **任务添加**：支持选择颜色、输入任务名称和设置时间范围
- **实时更新**：添加任务后，环状图和任务列表会自动更新
- **任务管理**：可以查看和删除已添加的任务
- **响应式设计**：适配各种屏幕尺寸

## 使用方法

1. 双击 `index.html` 文件在浏览器中打开应用
2. 点击右侧"已完成任务"旁边的加号按钮
3. 在弹出的表单中：
   - 选择任务颜色
   - 输入任务名称
   - 设置任务的开始和结束时间
4. 按回车键保存任务
5. 任务将同时显示在环状图（对应时间区域变色）和右侧任务列表中
6. 要删除任务，点击任务条目右侧的删除按钮

## 技术栈

- HTML5
- JavaScript
- Tailwind CSS v3
- Font Awesome

## 文件结构

- `index.html` - 应用的HTML结构和样式
- `script.js` - 应用的交互逻辑和环状图绘制
- `README.md` - 应用说明文档

## 注意事项

- 任务时间范围必须是有效的（结束时间晚于开始时间）
- 推荐使用现代浏览器（Chrome、Firefox、Edge等）获得最佳体验
